## API

| apis   | urls          | actions                |
| ------ | ------------- | ---------------------- |
| get    | api/users     | get all users          |
| get    | api/users/:id | get all users          |
| post   | api/users     | add new user data      |
| put    | api/users/:id | update user data       |
| delete | api/users/:id | remove an entity by id |
